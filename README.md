# CPU Scheduling Simulator

A Java Swing based desktop application that simulates common CPU scheduling algorithms used in Operating Systems.

## Features

- FCFS (First Come First Serve)
- SJF (Shortest Job First)
- Priority Scheduling
- Round Robin Scheduling
- Waiting Time Calculation
- Turnaround Time Calculation
- Completion Time Analysis
- Gantt Chart Visualization
- User-friendly GUI using Java Swing

## Technologies Used

- Java
- Java Swing
- OOP Concepts

## How to Run

1. Open project in IntelliJ IDEA / Eclipse
2. Run `Main.java`

## Project Purpose

This project was developed to understand how CPU scheduling algorithms work and compare their performance metrics.
